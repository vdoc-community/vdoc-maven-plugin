Full name

com.vdoc.maven:vdoc-maven-plugin:{{site.vdoc_maven_plugin_version}}:upload-setup

Attributes

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: deploy.

Parameters

buildDirectory
Directory containing the generated JAR.
  • java.io.File
  • ${project.build.directory}
jarName
Name of the generated JAR.
  • java.lang.String
  • ${project.build.finalName}
password
the user's password
  • java.lang.String
project
The Maven project.
  • org.apache.maven.project.MavenProject
  • ${project}
session
The Maven session.
  • org.apache.maven.execution.MavenSession
  • ${session}
setupName
where the setup can be found
  • java.lang.String
storeUrlHost
the restFull store url
  • java.lang.String
storeUrlport
the http port
  • int
  • 80
username
the user name with upload right
  • java.lang.String
vdocVersion
the vdoc version to build correct store path
  • java.lang.String
